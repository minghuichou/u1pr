# Your first pull request

本仓库是让你练习你在GitHub上的第一个pull request

首先fork本仓库

然后git clone你fork下来的仓库地址

``` 
git clone https://github.com/BaiYu96/u1pr.git
```

修改完成之后

* git add
* git commit -m "你的提交信息"
* git push

提交PR

当你的PR被我成功合并之后，你会在下面参与PR的小伙伴里面看到你的名字。
